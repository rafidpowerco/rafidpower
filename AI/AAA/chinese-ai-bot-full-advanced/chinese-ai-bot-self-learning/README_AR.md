## 🚀 البدء السريع

### المتطلبات:

- Node.js 18+
- MySQL 8.0+
- Python 3.10+ (اختياري - للأكواد المحلية)

### خطوات التثبيت:

```bash
# 1. استنساخ المشروع (إن كان داخل مستودع أكبر، ادخل مسار المجلد الفعلي)
git clone <repo-url>
cd chinese-ai-bot-self-learning

# 2. تثبيت الحزم
pnpm install

# 3. البيئة — انسخ القالب ثم عيّن DATABASE_URL وJWT_SECRET على الأقل
copy .env.example .env   # في PowerShell؛ أو: cp .env.example .env

# 4. ترحيل قاعدة البيانات (بعد إنشاء قاعدة MySQL)
pnpm db:migrate

# 5. التحقق الكامل قبل التطوير/النشر
pnpm run verify

# 6. التشغيل المحلي
pnpm dev
```

> عند تعديل **`drizzle/schema.ts`** استخدم `drizzle-kit` لتوليد ترحيلات جديدة؛ للتثبيت الأول يكفي **`pnpm db:migrate`** على الملفات الموجودة في `drizzle/`.

## 🔑 المتغيرات المطلوبة

أضف هذه المتغيرات إلى ملف `.env`:

```env
# Telegram
TELEGRAM_BOT_TOKEN=your_token_here

# OpenRouter (للنماذج المجانية)
OPENROUTER_API_KEY=your_key_here

# Database
DATABASE_URL=mysql://user:password@localhost:3306/bot_db

# Owner
OWNER_TELEGRAM_ID=your_id_here
```
