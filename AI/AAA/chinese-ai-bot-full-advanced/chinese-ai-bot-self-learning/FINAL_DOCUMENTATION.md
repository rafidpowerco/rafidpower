## 🚀 البدء السريع

### 1️⃣ **التثبيت**

```bash
cd chinese-ai-bot-self-learning
pnpm install
```

### 2️⃣ **إضافة المفاتيح**

أضف المفاتيح التالية في قسم Secrets:

- `TELEGRAM_BOT_TOKEN` - رمز بوت تليجرام
- `OPENROUTER_API_KEY` - مفتاح OpenRouter
- `OWNER_TELEGRAM_ID` - معرف تليجرام الخاص بك

### 3️⃣ **تشغيل البوت**

```bash
pnpm dev
```

### 4️⃣ **الوصول إلى الواجهات**

- **Telegram**: ابحث عن البوت واكتب `/start`
- **لوحة التحكم**: `http://localhost:3000/dashboard`
- **المحادثة**: `http://localhost:3000/chat`
- **الإحصائيات**: `http://localhost:3000/statistics`

## 🔌 API Endpoints

### المحادثات

- `POST /api/trpc/bot.sendMessage` - إرسال رسالة
- `GET /api/trpc/bot.getConversations` - الحصول على المحادثات
- `GET /api/trpc/bot.getConversation` - الحصول على محادثة محددة

### النماذج

- `GET /api/trpc/bot.getModels` - الحصول على قائمة النماذج
- `POST /api/trpc/bot.switchModel` - تبديل النموذج
- `GET /api/trpc/bot.getModelStats` - إحصائيات النموذج

### الإحصائيات

- `GET /api/trpc/bot.getStats` - الإحصائيات العامة
- `GET /api/trpc/bot.getInsights` - الرؤى والتوصيات
- `GET /api/trpc/bot.getReport` - التقرير الشامل

## 🧠 نظام التعلم الذاتي

### كيف يتعلم البوت؟

1. **تحليل المحادثات**: يحلل كل محادثة ويستخرج الأنماط
2. **تتبع الأداء**: يسجل نجاح/فشل كل مهمة
3. **اكتشاف الأنماط**: يكتشف الأنماط المتكررة
4. **التحسين المستمر**: يحسن الاختيارات بناءً على التجارب السابقة

### الأنماط المكتشفة

- **أنماط الأسئلة**: الأسئلة المتكررة والإجابات الفعالة
- **تفضيلات المستخدم**: الأسلوب المفضل والنماذج المفضلة
- **أنماط الأخطاء**: الأخطاء الشائعة والحلول
- **استراتيجيات النجاح**: الطرق الفعالة لحل المشاكل

## 🔧 التطوير والتخصيص

### إضافة نموذج جديد

```typescript
// في server/tools/toolManager.ts
const newTool: Tool = {
  id: "new_model",
  name: "New Model",
  type: "llm",
  provider: "Provider",
  description: "Description",
  capabilities: ["text_generation"],
  isAvailable: true,
  isLocal: false,
  rateLimit: { requestsPerMinute: 20, requestsPerDay: 200 },
  costPerRequest: 0,
  averageResponseTime: 2000,
  successRate: 0.95,
};

toolManager.addTool(newTool);
```

### إضافة ميزة جديدة

1. أنشئ ملف جديد في `server/features/`
2. أضف الوظائف المطلوبة
3. أضف API endpoint في `server/routers/botRouter.ts`
4. أضف واجهة في `client/src/pages/`

## 🐛 استكشاف الأخطاء

### المشكلة: البوت لا يرد على الرسائل

**الحل**: تحقق من:

- `TELEGRAM_BOT_TOKEN` صحيح
- الخادم يعمل بدون أخطاء
- الاتصال بـ OpenRouter يعمل

### المشكلة: أخطاء في قاعدة البيانات

**الحل**: تحقق من:

- `DATABASE_URL` صحيح
- قاعدة البيانات متاحة
- الهجرات تم تطبيقها

### المشكلة: بطء الاستجابة

**الحل**: جرب:

- استخدم نموذج أسرع
- تحقق من حالة الخادم
- قلل حجم المدخلات
