# rafidpower

## النشر (Render)

1. [Render Dashboard](https://dashboard.render.com) → **New** → **Web Service** → اختر المستودع `rafidpowerco/rafidpower`.
2. **Environment:** Docker (يستخدم `Dockerfile` في الجذر).
3. بعد **Deploy**، من **Settings** → **Custom Domains** أضف `rafidpower.xyz` وطبّق سجلات DNS في Spaceship كما تعرضها Render.
